 Overview

This repository contains solutions to C programming assignments that cover essential programming concepts including basic I/O operations, control structures, loops, functions, and pattern printing. These assignments are designed to build a strong foundation in C programming.

 Assignment Sets

 Assignment 1: Basic Input/Output and Arithmetic Operations
- Display messages
- Basic arithmetic operations (sum, average, addition, subtraction, multiplication, division)
- ASCII value display
- Simple interest calculation
- Temperature conversion (Fahrenheit to Celsius)
- Area of circle calculation

 Assignment 2: Conditional Statements and Decision Making
- Even/odd number checker
- Finding greatest of three numbers
- Leap year determination
- Character type identification (capital, small, digit, special symbol)
- Income tax calculator
- Day of week using switch-case

 Assignment 3: Loops and Pattern Printing
- Factorial calculation
- Power calculation (X raised to Y)
- Number reversal
- Prime number checker
- Sum of digits
- Various pattern printing programs (stars and numbers)

 Assignment 4: Functions
- Factorial using functions
- Maximum of two numbers
- Prime number checker using functions
- Number reversal using functions
- Area of circle using functions
- Fibonacci series
- Temperature conversion (Celsius to Fahrenheit)
- Menu-driven calculator
- Series calculation (1/1! + 2/2! + 3/3! + ...)
- Armstrong numbers (1-500)

 Prerequisites

- GCC compiler (recommended) or any C compiler
- Basic understanding of C programming syntax
- Command line/terminal access

 How to Compile and Run

On Linux/Mac:
```bash
# Compile
gcc filename.c -o output

# Run
./output
```

 On Windows:
```bash
# Compile
gcc filename.c -o output.exe

# Run
output.exe
```

Using online compilers:
You can also use online C compilers like:
- [OnlineGDB](https://www.onlinegdb.com/online_c_compiler)
- [Programiz](https://www.programiz.com/c-programming/online-compiler/)
- [Replit](https://replit.com/)
